Praias de Florianópolis

Florianópolis é um lugar democrático quando o assunto é praia: grande, pequena, para surfe, para crianças, para descansar, para badalar — tem de tudo. Costuma-se dizer que a Ilha da Magia tem 42 praias, mas os nativos falam que esse número é muito maior, chegando a afirmar que são mais de 100 praias. Os números são difíceis de confirmar e a praia mais bonita é difícil de eleger, mas é certo dizer que as opções são inúmeras e agradam os mais variados gostos.
